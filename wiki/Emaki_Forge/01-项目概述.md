# Emaki Forge - 项目概述

## 插件简介

**Emaki Forge** 是一款专为 Minecraft Paper 服务器设计的自定义锻造系统插件。它提供了完整的锻造工作流，包括图纸系统、材料管理、配方配置、品质系统和锻造历史记录。通过高度可配置的 YAML 文件，服主可以轻松创建独特的锻造体验，支持多种物品来源（原版、NeigeItems、CraftEngine），为玩家带来丰富的装备定制乐趣。

### 核心价值

- **灵活的锻造系统**：支持图纸、材料、配方的自由组合
- **多物品源支持**：无缝集成 NeigeItems、CraftEngine 等自定义物品插件
- **品质系统**：可配置的品质等级和保底机制
- **可视化界面**：直观的 GUI 界面和配方图鉴
- **高度可配置**：所有功能均可通过 YAML 文件自定义

## 功能特性

### 锻造系统

- **图纸系统**：定义锻造所需的图纸类型和标签
- **材料系统**：配置锻造材料及其属性贡献
- **配方系统**：创建锻造配方，定义输入输出
- **品质系统**：随机品质生成，支持保底机制

### 用户界面

- **锻造台 GUI**：直观的锻造操作界面
- **配方图鉴**：浏览所有可用配方
- **实时预览**：锻造前预览结果物品

### 数据管理

- **玩家数据存储**：记录锻造历史
- **自动保存**：可配置的自动保存间隔
- **数据持久化**：服务器重启后数据不丢失

## 模块架构

```
Emaki Forge
├── config/                    # 配置模型
│   └── AppConfig              # 主配置
│
├── loader/                    # 数据加载器
│   ├── BlueprintLoader        # 图纸加载器
│   ├── MaterialLoader         # 材料加载器
│   ├── RecipeLoader           # 配方加载器
│   ├── GuiTemplateLoader      # GUI模板加载器
│   └── PlayerDataStore        # 玩家数据存储
│
├── model/                     # 数据模型
│   ├── Blueprint              # 图纸定义
│   ├── ForgeMaterial          # 材料定义
│   ├── Recipe                 # 配方定义
│   ├── PlayerData             # 玩家数据
│   └── QualitySettings        # 品质设置
│
└── service/                   # 服务层
    ├── ForgeService           # 锻造核心服务
    ├── ForgeGuiService        # 锻造GUI服务
    ├── RecipeBookGuiService   # 配方图鉴服务
    ├── ItemIdentifierService  # 物品识别服务
    ├── ForgePdcService        # PDC数据服务
    ├── MessageService         # 消息服务
    └── BootstrapService       # 启动服务
```

## 快速上手

### 安装步骤

1. **安装前置插件**：确保已安装 Emaki CoreLib
2. **下载插件**：将 `Emaki_Forge-x.x.x.jar` 放入 `plugins` 目录
3. **启动服务器**：首次启动会自动生成默认配置
4. **验证安装**：使用 `/jiuwuforge help` 查看帮助

### 配置文件结构

```
plugins/Emaki_Forge/
├── config.yml                 # 主配置文件
├── lang/
│   └── zh_CN.yml              # 中文语言文件
├── blueprints/                # 图纸配置目录
│   ├── universal_blueprint.yml
│   └── weapon_sword.yml
├── materials/                 # 材料配置目录
│   ├── iron_essence.yml
│   └── flame_crystal.yml
├── recipes/                   # 配方配置目录
│   └── flame_sword.yml
└── gui/                       # GUI模板目录
    ├── forge_gui.yml
    └── recipe_book.yml
```

## 命令与权限

### 命令列表

| 命令 | 说明 | 权限 |
|------|------|------|
| `/jiuwuforge` | 打开通用锻造台 | `jiuwuforge.use` |
| `/jiuwuforge forge` | 打开通用锻造台 | `jiuwuforge.use` |
| `/jiuwuforge book` | 打开配方图鉴 | `jiuwuforge.book` |
