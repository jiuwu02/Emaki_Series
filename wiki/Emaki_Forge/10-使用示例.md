# Emaki Forge - 使用示例

## 快速开始

### 第一个配方

创建一个简单的铁剑锻造配方：

**1. 创建图纸配置** (`blueprints/weapon_sword.yml`)

```yaml
id: "weapon_sword"
display_name: "<gold>武器锻造图纸</gold>"
description:
  - "<gray>用于锻造武器类装备"

item: "ni-blueprint_weapon_sword"

tags:
  - "weapon"
  - "sword"

forge_capacity: 30
```

**2. 创建材料配置** (`materials/iron_essence.yml`)

```yaml
id: "iron_essence"
display_name: "<white>铁精粹</white>"
description:
  - "<gray>蕴含铁之精华的材料"

item: "ni-material_iron_essence"

capacity_cost: 5

priority: 10

effects:
  - type: "stat_contribution"
    stats:
      physical_damage:
        type: "constant"
        value: 5
```

**3. 创建配方配置** (`recipes/iron_sword.yml`)

```yaml
id: "iron_sword"
display_name: "<white>铁剑锻造配方</white>"

target_item:
  item: "v-IRON_SWORD"
  forge_capacity: 30

blueprint_requirements:
  - requirement_mode: "any_of"
    blueprint_options:
      - selector:
          kind: "id"
          value: "weapon_sword"
        count: 1

required_materials:
  - id: "iron_essence"
    count: 2

optional_materials:
  enabled: true
  whitelist: []
  blacklist: []
  max_count: 3

condition_type: "all_of"
conditions:
  - "%player_level% >= 5"

quality:
  enabled: true
  guarantee:
    enabled: false

result:
  output_item:
    item: "v-IRON_SWORD"
  success_message: "forge.success.crafted"

permission: "jiuwuforge.recipe.iron_sword"
```

**4. 重载配置**

```
/jiuwuforge reload
```

## API 使用示例

### 监听锻造完成

```java
public class ForgeListener implements Listener {
    
    private final EmakiForgePlugin forgePlugin;
    
    public ForgeListener(EmakiForgePlugin forgePlugin) {
        this.forgePlugin = forgePlugin;
    }
    
    @EventHandler
    public void onInventoryClose(InventoryCloseEvent event) {
        if (!(event.getPlayer() instanceof Player player)) return;
        
        ForgeGuiService service = forgePlugin.forgeGuiService();
        ForgeGuiSession session = service.getSession(player);
        
        if (session != null && session.forgeCompleted()) {
            // 锻造完成后的处理
            Recipe recipe = session.recipe();
            plugin.getLogger().info(player.getName() + " 完成了锻造: " + recipe.id());
        }
    }
}
```

### 自定义锻造检查

```java
public class CustomForgeChecker {
    
    private final EmakiForgePlugin forgePlugin;
    
    public boolean canPlayerForge(Player player, String recipeId) {
        // 获取配方
        Recipe recipe = forgePlugin.recipeLoader().getRecipe(recipeId);
        if (recipe == null) return false;
        
        // 检查权限
        if (recipe.requiresPermission() && 
            !player.hasPermission(recipe.permission())) {
            return false;
        }
        
        // 检查条件
        ConditionEvaluator evaluator = forgePlugin.conditionEvaluator();
        ConditionResult result = evaluator.evaluate(
            recipe.conditionType(),
            recipe.conditions(),
            player
        );
        
        return result.success();
    }
}
```

### 批量锻造

```java
public class BatchForgeService {
    
    private final EmakiForgePlugin forgePlugin;
    
    public void batchForge(Player player, String recipeId, int count) {
        Recipe recipe = forgePlugin.recipeLoader().getRecipe(recipeId);
        if (recipe == null) return;
        
        ForgeService forgeService = forgePlugin.forgeService();
        MessageService messageService = forgePlugin.messageService();
        
        int success = 0;
        int failed = 0;
        
        for (int i = 0; i < count; i++) {
            // 构建虚拟 GUI 物品
            ForgeService.GuiItems guiItems = buildGuiItems(recipe);
            
            // 执行锻造
            ForgeService.ForgeResult result = forgeService.executeForge(
                player, recipe, guiItems
            );
            
            if (result.success()) {
                success++;
                // 给予物品
                player.getInventory().addItem(result.resultItem());
            } else {
                failed++;
            }
        }
        
        messageService.send(player, "batch_forge.complete", Map.of(
            "success", success,
            "failed", failed
        ));
    }
    
    private ForgeService.GuiItems buildGuiItems(Recipe recipe) {
        // 构建必需材料
        Map<Integer, ItemStack> requiredMaterials = new HashMap<>();
        int slot = 0;
        for (Recipe.RequiredMaterial mat : recipe.requiredMaterials()) {
            ItemStack item = forgePlugin.itemIdentifierService()
                .createItem(mat.source(), mat.count());
            requiredMaterials.put(slot++, item);
        }
        
        return new ForgeService.GuiItems(
            null,  // 无目标物品
            new HashMap<>(),  // 图纸
            requiredMaterials,
            new HashMap<>()  // 无可选材料
        );
    }
}
```

### 查询玩家锻造历史

```java
public class ForgeHistoryChecker {
    
    private final EmakiForgePlugin forgePlugin;
    
    public void showHistory(Player player) {
        PlayerDataStore dataStore = forgePlugin.playerDataStore();
        
        player.sendMessage("<gold>═══ 锻造历史 ═══</gold>");
        
        for (Recipe recipe : forgePlugin.recipeLoader().getAllRecipes()) {
            int count = dataStore.getCraftCount(player.getUniqueId(), recipe.id());
            if (count > 0) {
                player.sendMessage("<gray>" + recipe.displayName() + 
                    ": <yellow>" + count + " 次");
            }
        }
    }
    
    public boolean hasCraftedLegend(Player player) {
        PlayerDataStore dataStore = forgePlugin.playerDataStore();
        
        for (Recipe recipe : forgePlugin.recipeLoader().getAllRecipes()) {
            if (recipe.id().contains("legendary") || 
                recipe.id().contains("dragon")) {
                if (dataStore.hasCrafted(player.getUniqueId(), recipe.id())) {
                    return true;
                }
            }
        }
        return false;
    }
}
```
