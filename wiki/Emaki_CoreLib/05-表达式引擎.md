# Emaki CoreLib - 表达式引擎

## 概述

表达式引擎（ExpressionEngine）提供了强大的数学表达式计算能力，支持变量替换、自定义函数和多种随机分布。该模块基于 exp4j 库构建，为插件提供灵活的数值计算和配置能力。

---

## 核心功能

### 基础表达式计算

```java
// 基础算术
double result1 = ExpressionEngine.evaluate("2 + 3 * 4");        // 14.0
double result2 = ExpressionEngine.evaluate("(2 + 3) * 4");      // 20.0

// 使用变量
double result3 = ExpressionEngine.evaluate(
    "{level} * 10 + {base}",
    Map.of("level", 5, "base", 100)
);  // 150.0
```

### 支持的运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法 | `2 + 3` = 5 |
| `-` | 减法 | `5 - 2` = 3 |
| `*` | 乘法 | `3 * 4` = 12 |
| `/` | 除法 | `10 / 2` = 5 |
| `%` | 取模 | `10 % 3` = 1 |
| `^` | 幂运算 | `2 ^ 3` = 8 |

### 内置函数

| 函数 | 说明 | 示例 |
|------|------|------|
| `ceil(x)` | 向上取整 | `ceil(3.2)` = 4 |
| `floor(x)` | 向下取整 | `floor(3.8)` = 3 |
| `round(x)` | 四舍五入 | `round(3.5)` = 4 |
| `log10(x)` | 以10为底的对数 | `log10(100)` = 2 |
| `min(a, b)` | 最小值 | `min(3, 5)` = 3 |
| `max(a, b)` | 最大值 | `max(3, 5)` = 5 |
| `pow(a, b)` | 幂运算 | `pow(2, 3)` = 8 |
| `sqrt(x)` | 平方根 | `sqrt(16)` = 4 |
| `abs(x)` | 绝对值 | `abs(-5)` = 5 |
| `sin(x)` | 正弦 | `sin(0)` = 0 |
| `cos(x)` | 余弦 | `cos(0)` = 1 |
| `tan(x)` | 正切 | `tan(0)` = 0 |

## 变量系统

### 变量格式

变量使用花括号包裹：`{variable_name}`

### 使用示例

```java
Map<String, Object> variables = Map.of(
    "player_level", 50,
    "base_damage", 100,
    "multiplier", 1.5
);

double damage = ExpressionEngine.evaluate(
    "{base_damage} * (1 + {player_level} / 100) * {multiplier}",
    variables
);
// 结果: 100 * 1.5 * 1.5 = 225.0
```

### 变量替换规则

- 变量名区分大小写
- 未定义的变量替换为 0
- 支持嵌套表达式

## 随机分布系统

### evaluateRandomConfig() 方法

```java
public static double evaluateRandomConfig(Object config)
```

该方法支持多种随机分布配置，是表达式引擎的核心功能之一。

---

### 1. 常量类型 (constant)

固定值，不产生随机性。

```yaml
# YAML 配置
value:
  type: "constant"
  value: 100

# 简写形式
value: 100
```

```java
double result = ExpressionEngine.evaluateRandomConfig(config);
// 结果始终为 100.0
```

---

### 2. 均匀分布 (uniform)

在指定范围内均匀随机取值。

```yaml
value:
  type: "uniform"
  min: 10
  max: 20
```

```java
// 结果在 [10, 20) 范围内均匀分布
double result = ExpressionEngine.evaluateRandomConfig(config);
```

**简写形式**：

```yaml
# 使用波浪号表示范围
value: "10 ~ 20"
```

---

### 3. 高斯分布 (gaussian)

正态分布，以均值为中心产生随机值。

```yaml
value:
  type: "gaussian"
  mean: 100        # 均值
  std_dev: 15      # 标准差
  min: 50          # 最小值 (可选)
  max: 150         # 最大值 (可选)
  max_attempts: 128  # 最大尝试次数
```

**参数说明**：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `mean` | Double | 否 | (min+max)/2 | 均值 |
| `std_dev` | Double | 否 | (max-min)/6 | 标准差 |
| `min` | Double | 否 | null | 最小值限制 |
| `max` | Double | 否 | null | 最大值限制 |
| `max_attempts` | Integer | 否 | 128 | 超出范围时的最大重试次数 |

**分布特性**：
- 约 68% 的值在 [mean - std_dev, mean + std_dev] 范围内
- 约 95% 的值在 [mean - 2*std_dev, mean + 2*std_dev] 范围内

---

### 4. 偏态正态分布 (skew_normal)

支持偏斜的正态分布，可以产生不对称的分布曲线。

```yaml
value:
  type: "skew_normal"
  mean: 100
  std_dev: 15
  skewness: 0.5    # 偏度 (-0.995 到 0.995)
  min: 50
  max: 150
```

**偏度说明**：
- `skewness > 0`：右偏（正偏），分布向右延伸
- `skewness < 0`：左偏（负偏），分布向左延伸
- `skewness = 0`：对称分布，等同于普通高斯分布

---

### 5. 三角分布 (triangle)

以 mode 为中心，deviation 为半宽的三角分布。

```yaml
value:
  type: "triangle"
  mode: 100        # 中心值
  deviation: 20    # 偏差范围
```

**分布范围**：[mode - deviation, mode + deviation]

**特性**：mode 处概率最高，向两侧线性递减

---

### 6. 表达式类型 (expression)

直接计算数学表达式。

```yaml
value:
  type: "expression"
  expression: "{base} * 1.5 + 10"
```

## 完整配置示例

### 武器伤害配置

```yaml
weapon:
  base_damage:
    type: "constant"
    value: 50
  
  random_bonus:
    type: "uniform"
    min: 5
    max: 15
  
  critical_multiplier:
    type: "gaussian"
    mean: 1.5
    std_dev: 0.2
    min: 1.0
    max: 2.5
  
  quality_factor:
    type: "skew_normal"
    mean: 1.0
    std_dev: 0.15
    skewness: 0.3  # 偏向高品质
    min: 0.8
    max: 1.5
```

### 解析与计算

```java
public class WeaponGenerator {
    
    public double calculateDamage(ConfigurationSection config, int playerLevel) {
        double baseDamage = ExpressionEngine.evaluateRandomConfig(config.get("base_damage"));
        double randomBonus = ExpressionEngine.evaluateRandomConfig(config.get("random_bonus"));
        double critMultiplier = ExpressionEngine.evaluateRandomConfig(config.get("critical_multiplier"));
        double qualityFactor = ExpressionEngine.evaluateRandomConfig(config.get("quality_factor"));
        
        // 最终伤害计算
        return ExpressionEngine.evaluate(
            "{base} + {bonus} * {crit} * {quality} * (1 + {level} / 100)",
            Map.of(
                "base", baseDamage,
                "bonus", randomBonus,
                "crit", critMultiplier,
                "quality", qualityFactor,
                "level", playerLevel
            )
        );
    }
}
```

## 嵌套配置

随机配置支持嵌套，实现复杂的随机逻辑：

```yaml
complex_value:
  type: "uniform"
  min:
    type: "gaussian"
    mean: 10
    std_dev: 2
  max:
    type: "constant"
    value: 50
```

```java
// 先随机生成 min 值，再在 [min, 50] 范围内均匀随机
double result = ExpressionEngine.evaluateRandomConfig(config);
```

## 使用示例

### 示例1：等级缩放伤害

```java
public double calculateScaledDamage(int level, double baseDamage) {
    return ExpressionEngine.evaluate(
        "{base} * (1 + 0.1 * {level}) * pow(1.05, {level})",
        Map.of("base", baseDamage, "level", level)
    );
}
```

### 示例2：随机属性生成

```yaml
# equipment_attributes.yml
attributes:
  strength:
    type: "gaussian"
    mean: 50
    std_dev: 10
    min: 20
    max: 80
  
  agility:
    type: "skew_normal"
    mean: 40
    std_dev: 8
    skewness: -0.3  # 偏向低敏捷
    min: 15
    max: 60
  
  luck:
    type: "uniform"
    min: 1
    max: 100
```

```java
public Map<String, Double> generateAttributes(ConfigurationSection config) {
    Map<String, Double> attributes = new HashMap<>();
    
    for (String attrName : config.getKeys(false)) {
        double value = ExpressionEngine.evaluateRandomConfig(config.get(attrName));
        attributes.put(attrName, value);
    }
    
    return attributes;
}
```

### 示例3：条件概率计算

```java
public boolean rollSuccess(double baseChance, int luckStat) {
    double finalChance = ExpressionEngine.evaluate(
        "{base} + {luck} * 0.001",
        Map.of("base", baseChance, "luck", luckStat)
    );
    
    return Randoms.chance(Math.min(finalChance, 1.0));
}
```

## 错误处理

### 表达式错误

当表达式无效时，`evaluate()` 返回 0.0：

```java
double result = ExpressionEngine.evaluate("invalid + expression");  // 返回 0.0
```

### 变量缺失

未定义的变量替换为空字符串，可能导致表达式无效：

```java
double result = ExpressionEngine.evaluate("{undefined} + 10");  // 可能返回 0.0
```

### 最佳实践

```java
public double safeEvaluate(String expression, Map<String, ?> variables) {
    if (Texts.isBlank(expression)) {
        return 0.0;
    }
    
    try {
        return ExpressionEngine.evaluate(expression, variables);
    } catch (Exception e) {
        plugin.getLogger().warning("表达式计算失败: " + expression + ", 错误: " + e.getMessage());
        return 0.0;
    }
}
```

## 性能优化

### 1. 缓存编译后的表达式

exp4j 内部会缓存编译后的表达式，重复计算相同表达式时性能较好。

### 2. 避免频繁解析配置

对于频繁使用的随机配置，建议预解析：

```java
private Object cachedDamageConfig;

public void loadConfig(ConfigurationSection section) {
    cachedDamageConfig = section.get("damage");
}

public double rollDamage() {
    return ExpressionEngine.evaluateRandomConfig(cachedDamageConfig);
}
```
