# Emaki CoreLib - 数学工具

## 概述

数学工具模块提供了数值处理和随机数生成的实用工具类，包括 `Numbers`（数值处理）和 `Randoms`（随机数生成）两个核心类。

---

## Numbers - 数值处理

### 核心方法

#### tryParseInt() - 安全解析整数

```java
public static int tryParseInt(Object value, int defaultValue)
public static Integer tryParseInt(Object value, Integer defaultValue)
```

**参数**：
- `value` - 待解析的值（可以是 String、Number 等）
- `defaultValue` - 解析失败时的默认值

**返回值**：解析后的整数值

**示例**：

```java
int value1 = Numbers.tryParseInt("123", 0);           // 123
int value2 = Numbers.tryParseInt("abc", 0);           // 0 (默认值)
int value3 = Numbers.tryParseInt(45.67, 0);           // 45
int value4 = Numbers.tryParseInt(null, 10);           // 10 (默认值)
```

---

#### tryParseDouble() - 安全解析浮点数

```java
public static double tryParseDouble(Object value, double defaultValue)
public static Double tryParseDouble(Object value, Double defaultValue)
```

**示例**：

```java
double value1 = Numbers.tryParseDouble("3.14", 0.0);  // 3.14
double value2 = Numbers.tryParseDouble("invalid", 0.0);  // 0.0
double value3 = Numbers.tryParseDouble(100, 0.0);     // 100.0
```

---

#### isNumeric() - 数值检查

```java
public static boolean isNumeric(String text)
```

**示例**：

```java
Numbers.isNumeric("123");      // true
Numbers.isNumeric("3.14");     // true
Numbers.isNumeric("-50");      // true
Numbers.isNumeric("abc");      // false
Numbers.isNumeric("");         // false
Numbers.isNumeric(null);       // false
```

---

#### clamp() - 数值限制

```java
public static int clamp(int value, int min, int max)
public static double clamp(double value, double min, double max)
```

**说明**：将值限制在指定范围内

**示例**：

```java
int result1 = Numbers.clamp(50, 0, 100);   // 50
int result2 = Numbers.clamp(-10, 0, 100);  // 0 (低于最小值)
int result3 = Numbers.clamp(150, 0, 100);  // 100 (高于最大值)
```

---

#### roundToInt() - 四舍五入

```java
public static int roundToInt(double value)
```

**示例**：

```java
int result1 = Numbers.roundToInt(3.4);   // 3
int result2 = Numbers.roundToInt(3.5);   // 4
int result3 = Numbers.roundToInt(3.6);   // 4
```

## Randoms - 随机数生成

### 基础随机方法

#### randomInt() - 随机整数

```java
public static int randomInt(int min, int max)
```

**说明**：生成 [min, max] 范围内的随机整数（包含边界）

**示例**：

```java
int dice = Randoms.randomInt(1, 6);  // 模拟骰子: 1-6
int level = Randoms.randomInt(10, 20);  // 10-20 之间的随机等级
```

---

#### randomDouble() - 随机浮点数

```java
public static double randomDouble()
```

**说明**：生成 [0.0, 1.0) 范围内的随机浮点数

**示例**：

```java
double chance = Randoms.randomDouble();  // 0.0 到 1.0 之间
```

---

#### chance() - 概率判断

```java
public static boolean chance(double probability)
```

**参数**：
- `probability` - 概率值 (0.0-1.0)

**返回值**：以指定概率返回 true

**示例**：

```java
// 50% 概率
if (Randoms.chance(0.5)) {
    player.sendMessage("幸运！你获得了额外奖励！");
}

// 10% 暴击概率
if (Randoms.chance(0.1)) {
    damage *= 2;  // 暴击双倍伤害
}
```

---

### 分布随机方法

#### uniform() - 均匀分布

```java
public static double uniform(double min, double max)
```

**说明**：在 [min, max) 范围内均匀分布随机取值

**示例**：

```java
double damage = Randoms.uniform(10, 20);  // 10-20 之间的随机伤害
```

---

#### gaussian() - 高斯分布

```java
public static double gaussian(double mean, double stdDev, Double min, Double max, int maxAttempts)
```

**参数**：
- `mean` - 均值
- `stdDev` - 标准差
- `min` - 最小值限制（可选）
- `max` - 最大值限制（可选）
- `maxAttempts` - 超出范围时的最大重试次数

**示例**：

```java
// 生成符合正态分布的属性值
double strength = Randoms.gaussian(50, 10, 20, 80, 128);
// 均值50，标准差10，范围限制在20-80
```

---

#### skewNormal() - 偏态正态分布

```java
public static double skewNormal(double mean, double stdDev, double skewness, Double min, Double max, int maxAttempts)
```

**参数**：
- `skewness` - 偏度 (-0.995 到 0.995)

**示例**：

```java
// 偏向高品质的属性生成
double quality = Randoms.skewNormal(1.0, 0.15, 0.3, 0.8, 1.5, 128);
```

---

#### triangle() - 三角分布

```java
public static double triangle(double mode, double deviation)
```

**参数**：
- `mode` - 中心值（概率最高）
- `deviation` - 偏差范围

**示例**：

```java
// 以100为中心，偏差20的三角分布
double value = Randoms.triangle(100, 20);  // 范围 [80, 120]
```

---

### 加权随机

#### weightedRandom() - 加权随机选择

```java
public static <T> T weightedRandom(List<Weighted<T>> items)
```

**Weighted 记录类**：

```java
public record Weighted<T>(T item, double weight) {}
```

**示例**：

```java
List<Randoms.Weighted<String>> rewards = List.of(
    new Randoms.Weighted<>("common", 60),    // 60% 概率
    new Randoms.Weighted<>("rare", 30),      // 30% 概率
    new Randoms.Weighted<>("legendary", 10)  // 10% 概率
);

String reward = Randoms.weightedRandom(rewards);
// 根据权重随机选择奖励
```

## 完整使用示例

### 示例1：装备属性生成

```java
public class EquipmentGenerator {
    
    public Map<String, Double> generateAttributes(int itemLevel) {
        Map<String, Double> attributes = new HashMap<>();
        
        // 基础属性随等级增长
        double baseMultiplier = 1 + itemLevel * 0.1;
        
        // 力量：高斯分布
        attributes.put("strength", Randoms.gaussian(
            10 * baseMultiplier,  // 均值
            2 * baseMultiplier,   // 标准差
            5.0,                  // 最小值
            20 * baseMultiplier,  // 最大值
            128                   // 最大尝试次数
        ));
        
        // 敏捷：均匀分布
        attributes.put("agility", Randoms.uniform(
            5 * baseMultiplier,
            15 * baseMultiplier
        ));
        
        // 幸运：偏态分布（偏向低值）
        attributes.put("luck", Randoms.skewNormal(
            5,      // 均值
            2,      // 标准差
            -0.3,   // 偏度（左偏）
            1.0,    // 最小值
            15.0,   // 最大值
            128
        ));
        
        return attributes;
    }
}
```

### 示例2：战利品系统

```java
public class LootSystem {
    
    public ItemStack generateLoot(Player player, MonsterType monsterType) {
        // 根据怪物类型确定战利品池
        List<Randoms.Weighted<LootEntry>> lootPool = getLootPool(monsterType);
        
        // 加权随机选择
        Randoms.Weighted<LootEntry> selected = Randoms.weightedRandom(lootPool);
        if (selected == null) return null;
        
        LootEntry entry = selected.item();
        
        // 随机数量
        int amount = Randoms.randomInt(entry.minAmount(), entry.maxAmount());
        
        // 随机品质
        double quality = Randoms.chance(0.1) ? 1.5 : 1.0;
        
        return createItem(entry.itemId(), amount, quality);
    }
    
    private List<Randoms.Weighted<LootEntry>> getLootPool(MonsterType type) {
        return switch (type) {
            case BOSS -> List.of(
                new Randoms.Weighted<>(new LootEntry("legendary_sword", 1, 1), 5),
                new Randoms.Weighted<>(new LootEntry("rare_gem", 1, 3), 20),
                new Randoms.Weighted<>(new LootEntry("gold_coin", 100, 500), 75)
            );
            case ELITE -> List.of(
                new Randoms.Weighted<>(new LootEntry("rare_material", 1, 2), 15),
                new Randoms.Weighted<>(new LootEntry("gold_coin", 50, 100), 85)
            );
            default -> List.of(
                new Randoms.Weighted<>(new LootEntry("common_material", 1, 5), 50),
                new Randoms.Weighted<>(new LootEntry("gold_coin", 10, 30), 50)
            );
        };
    }
}
```

### 示例3：技能伤害计算

```java
public class DamageCalculator {
    
    public double calculateDamage(Player attacker, LivingEntity target, Skill skill) {
        // 基础伤害
        double baseDamage = skill.baseDamage();
        
        // 攻击力加成
        double attackBonus = attacker.getAttribute(Attribute.GENERIC_ATTACK_DAMAGE).getValue();
        
        // 随机浮动 (±10%)
        double variance = Randoms.uniform(0.9, 1.1);
        
        // 暴击判定
        double critMultiplier = 1.0;
        if (Randoms.chance(getCritChance(attacker))) {
            critMultiplier = getCritMultiplier(attacker);
            attacker.sendMessage("§c暴击！");
        }
        
        // 最终伤害
        double finalDamage = baseDamage + attackBonus;
        finalDamage *= variance * critMultiplier;
        
        // 限制最小伤害
        return Numbers.clamp(finalDamage, 1.0, Double.MAX_VALUE);
    }
    
    private double getCritChance(Player player) {
        // 基础暴击率 5% + 敏捷加成
        return 0.05 + player.getAttribute(Attribute.GENERIC_LUCK).getValue() * 0.01;
    }
    
    private double getCritMultiplier(Player player) {
        // 暴击倍率：正态分布，均值1.5，标准差0.2
        return Randoms.gaussian(1.5, 0.2, 1.2, 2.5, 128);
    }
}
```

### 示例4：配置解析

```yaml
# config.yml
damage_settings:
  base_damage: 100
  variance:
    min: 0.9
    max: 1.1
  crit:
    base_chance: 0.05
    multiplier:
      mean: 1.5
      std_dev: 0.2
      min: 1.2
      max: 2.5
```

```java
public class DamageConfig {
    
    public double rollDamage(Player player, ConfigurationSection config) {
        double base = config.getDouble("base_damage");
        
        // 伤害浮动
        double min = config.getDouble("variance.min");
        double max = config.getDouble("variance.max");
        double variance = Randoms.uniform(min, max);
        
        // 暴击判定
        double critMultiplier = 1.0;
        double critChance = config.getDouble("crit.base_chance");
        
        if (Randoms.chance(critChance)) {
            ConfigurationSection critConfig = config.getConfigurationSection("crit.multiplier");
            critMultiplier = Randoms.gaussian(
                critConfig.getDouble("mean"),
                critConfig.getDouble("std_dev"),
                critConfig.getDouble("min"),
                critConfig.getDouble("max"),
                128
            );
        }
        
        return base * variance * critMultiplier;
    }
}
```

## 最佳实践

### 1. 使用 ThreadLocalRandom

`Randoms` 内部使用 `ThreadLocalRandom`，在多线程环境下性能更好：

```java
// Randoms 内部实现
ThreadLocalRandom.current().nextDouble(min, max);
```

### 2. 避免频繁创建 Random 对象

```java
// 不推荐
Random random = new Random();
double value = random.nextDouble();

// 推荐
double value = Randoms.randomDouble();
```

### 3. 数值边界检查

始终使用 `clamp` 确保值在有效范围内：

```java
double damage = Numbers.clamp(calculatedDamage, 0, maxHealth);
```
