# Emaki CoreLib - 物品源系统

## 概述

物品源系统（ItemSource System）提供了统一的物品标识和创建机制，支持多种物品来源类型，包括原版物品、NeigeItems 自定义物品和 CraftEngine 物品。该系统采用抽象工厂模式，让上层代码无需关心物品的具体来源。

---

## 核心概念

### ItemSource - 物品源

`ItemSource` 是物品源的抽象表示，包含两个核心属性：

```java
public final class ItemSource {
    private final ItemSourceType type;       // 物品类型
    private final String identifier;         // 物品标识符
}
```

### ItemSourceType - 物品类型

| 类型 | 枚举值 | 前缀 | 示例 |
|------|--------|------|------|
| 原版物品 | `VANILLA` | `v-`, `minecraft-` | `v-DIAMOND`, `minecraft:diamond` |
| NeigeItems | `NEIGEITEMS` | `ni-`, `neigeitems-` | `ni-custom_sword` |
| CraftEngine | `CRAFTENGINE` | `ce-`, `craftengine-` | `ce-namespace:item_id` |

---

## 简写语法

### 基本格式

```
前缀-标识符
```

### 示例

| 简写 | 解析结果 |
|------|---------|
| `DIAMOND` | VANILLA: DIAMOND |
| `v-DIAMOND` | VANILLA: DIAMOND |
| `minecraft-diamond` | VANILLA: DIAMOND |
| `ni-flame_sword` | NEIGEITEMS: flame_sword |
| `neigeitems-custom_item` | NEIGEITEMS: custom_item |
| `ce-weapon:blade` | CRAFTENGINE: weapon:blade |
| `craftengine-mod:gem` | CRAFTENGINE: mod:gem |

### 无前缀处理

当简写不包含前缀时，系统默认将其解析为原版物品：

```java
"DIAMOND"          → VANILLA: DIAMOND
"IRON_INGOT"       → VANILLA: IRON_INGOT
"minecraft:diamond" → VANILLA: DIAMOND (自动处理命名空间)
```

## ItemSourceRegistry - 物品源注册表

### 功能说明

`ItemSourceRegistry` 是物品源解析的中心枢纽，负责：
- 管理内置和自定义解析器
- 按优先级顺序尝试解析
- 提供统一的解析入口

### 获取系统注册表

```java
ItemSourceRegistry registry = ItemSourceRegistry.system();
```

### 解析物品源

```java
ItemSource source = registry.parseShorthand("ni-custom_sword");
```

### 注册自定义解析器

```java
registry.registerParser(shorthand -> {
    if (shorthand.startsWith("myplugin-")) {
        String id = shorthand.substring("myplugin-".length());
        return new ItemSource(MyPluginItemType.INSTANCE, id);
    }
    return null;  // 返回 null 表示不处理此简写
});
```

### 设置回退解析器

```java
registry.setFallbackParser(shorthand -> {
    // 自定义默认解析逻辑
    return new ItemSource(ItemSourceType.VANILLA, shorthand);
});
```

## ItemSourceUtil - 工具类

### 解析方法

#### parse() - 解析物品源

```java
public static ItemSource parse(Object raw)
```

**参数**：
- `raw` - 可以是字符串简写或 ConfigurationSection

**返回值**：
- 解析成功返回 `ItemSource`
- 解析失败返回 `null`

**示例**：

```java
// 字符串简写
ItemSource source1 = ItemSourceUtil.parse("ni-flame_sword");

// YAML 配置节点
ItemSource source2 = ItemSourceUtil.parse(configSection);
```

#### YAML 配置格式

```yaml
# 简写形式
item: "ni-flame_sword"

# 完整形式
item:
  type: "neigeitems"
  id: "flame_sword"

# 或使用简写键
ni-item: "flame_sword"
```

---

### 匹配方法

#### matches() - 匹配检查

```java
public static boolean matches(ItemStack itemStack, ItemSource source)
```

**参数**：
- `itemStack` - 待检查的物品
- `source` - 目标物品源

**返回值**：
- 匹配返回 `true`
- 不匹配返回 `false`

**示例**：

```java
ItemStack item = player.getInventory().getItemInMainHand();
ItemSource target = ItemSourceUtil.parse("ni-flame_sword");

if (ItemSourceUtil.matches(item, target)) {
    player.sendMessage("你持有烈焰之剑！");
}
```

---

### 转换方法

#### toShorthand() - 转换为简写

```java
public static String toShorthand(ItemSource source)
```

**返回值**：物品源的简写字符串表示

**示例**：

```java
ItemSource source = new ItemSource(ItemSourceType.NEIGEITEMS, "flame_sword");
String shorthand = ItemSourceUtil.toShorthand(source);
// 结果: "ni-flame_sword"
```

## 完整使用示例

### 示例1：创建物品

```java
public class ItemManager {
    
    private final ItemSourceRegistry registry = ItemSourceRegistry.system();
    
    public ItemStack createItem(String shorthand, int amount) {
        ItemSource source = registry.parseShorthand(shorthand);
        if (source == null) {
            return null;
        }
        
        return switch (source.getType()) {
            case VANILLA -> createVanillaItem(source.getIdentifier(), amount);
            case NEIGEITEMS -> createNeigeItemsItem(source.getIdentifier(), amount);
            case CRAFTENGINE -> createCraftEngineItem(source.getIdentifier(), amount);
        };
    }
    
    private ItemStack createVanillaItem(String id, int amount) {
        Material material = Material.matchMaterial(id);
        return material != null ? new ItemStack(material, amount) : null;
    }
    
    private ItemStack createNeigeItemsItem(String id, int amount) {
        return ItemManager.INSTANCE.getItem(id, null, amount);
    }
    
    private ItemStack createCraftEngineItem(String id, int amount) {
        Key key = Key.parse(id);
        CustomItem<?> item = CraftEngineItems.byId(key);
        return item != null ? item.buildItemStack(ItemBuildContext.empty(), amount) : null;
    }
}
```

### 示例2：配置文件解析

```yaml
# rewards.yml
rewards:
  daily:
    items:
      - "ni-daily_coin x10"
      - "v-DIAMOND x5"
      - "ce-gems:ruby x3"
  
  weekly:
    items:
      - item: "ni-legendary_chest"
        amount: 1
      - item: "minecraft:netherite_ingot"
        amount: 3
```

```java
public class RewardManager {
    
    public List<ItemStack> parseRewards(ConfigurationSection section) {
        List<ItemStack> items = new ArrayList<>();
        
        for (Object raw : section.getList("items")) {
            if (raw instanceof String shorthand) {
                // 解析 "ni-item_id x10" 格式
                String[] parts = shorthand.split(" x");
                ItemSource source = ItemSourceUtil.parse(parts[0]);
                int amount = parts.length > 1 ? Integer.parseInt(parts[1]) : 1;
                
                if (source != null) {
                    ItemStack item = createItem(source, amount);
                    if (item != null) {
                        items.add(item);
                    }
                }
            } else if (raw instanceof Map<?, ?> map) {
                // 解析完整配置格式
                ItemSource source = ItemSourceUtil.parse(map.get("item"));
                int amount = (Integer) map.getOrDefault("amount", 1);
                
                if (source != null) {
                    ItemStack item = createItem(source, amount);
                    if (item != null) {
                        items.add(item);
                    }
                }
            }
        }
        
        return items;
    }
}
```

### 示例3：物品识别与验证

```java
public class CraftingValidator {
    
    private final Map<String, ItemSource> requiredItems = new HashMap<>();
    
    public void loadRecipe(ConfigurationSection section) {
        for (String key : section.getKeys(false)) {
            ItemSource source = ItemSourceUtil.parse(section.get(key));
            if (source != null) {
                requiredItems.put(key, source);
            }
        }
    }
    
    public boolean validateInput(Player player, Inventory inventory) {
        Map<ItemSource, Integer> collected = new HashMap<>();
        
        for (ItemStack item : inventory.getContents()) {
            if (item == null || item.getType() == Material.AIR) continue;
            
            ItemSource identified = identifyItem(item);
            if (identified != null) {
                collected.merge(identified, item.getAmount(), Integer::sum);
            }
        }
        
        for (Map.Entry<String, ItemSource> entry : requiredItems.entrySet()) {
            int required = getRequiredCount(entry.getKey());
            int available = collected.getOrDefault(entry.getValue(), 0);
            
            if (available < required) {
                return false;
            }
        }
        
        return true;
    }
    
    private ItemSource identifyItem(ItemStack item) {
        // 尝试识别为 NeigeItems
        String niId = ItemManager.INSTANCE.getItemId(item);
        if (niId != null) {
            return new ItemSource(ItemSourceType.NEIGEITEMS, niId);
        }
        
        // 尝试识别为 CraftEngine
        Key ceKey = CraftEngineItems.idByItemStack(item);
        if (ceKey != null) {
            return new ItemSource(ItemSourceType.CRAFTENGINE, ceKey.toString());
        }
        
        // 默认为原版
        return new ItemSource(ItemSourceType.VANILLA, item.getType().name());
    }
}
```

## 扩展物品源类型

### 定义新类型

```java
public enum CustomItemType implements ItemSourceType {
    MY_PLUGIN;
    
    @Override
    public String getPrefix() {
        return "mp";
    }
}
```

### 注册解析器

```java
ItemSourceRegistry registry = ItemSourceRegistry.system();

registry.registerParser(shorthand -> {
    if (shorthand.startsWith("mp-")) {
        String id = shorthand.substring(3);
        return new ItemSource(CustomItemType.MY_PLUGIN, id);
    }
    return null;
});
```

### 创建物品

```java
public ItemStack createCustomItem(ItemSource source, int amount) {
    if (source.getType() == CustomItemType.MY_PLUGIN) {
        return MyPlugin.getItemManager().create(source.getIdentifier(), amount);
    }
    return null;
}
```

## 最佳实践

### 1. 统一使用简写

在配置文件中优先使用简写形式，提高可读性：

```yaml
# 推荐
item: "ni-flame_sword"

# 不推荐
item:
  type: "neigeitems"
  id: "flame_sword"
```

### 2. 空值检查

始终检查解析结果是否为 null：

```java
ItemSource source = ItemSourceUtil.parse(config.get("item"));
if (source == null) {
    plugin.getLogger().warning("无效的物品源配置: " + config.getCurrentPath());
    return;
}
```

### 3. 缓存物品源

对于频繁使用的物品源，建议缓存解析结果：

```java
private final Map<String, ItemSource> sourceCache = new ConcurrentHashMap<>();

public ItemSource getCachedSource(String shorthand) {
    return sourceCache.computeIfAbsent(shorthand, ItemSourceUtil::parse);
}
```

### 4. 错误处理

提供友好的错误提示：

```java
public ItemStack safeCreateItem(String shorthand, int amount) {
    ItemSource source = ItemSourceUtil.parse(shorthand);
    if (source == null) {
        plugin.getLogger().warning("无法解析物品源: " + shorthand);
        return null;
    }
    
    ItemStack item = createItem(source, amount);
    if (item == null) {
        plugin.getLogger().warning("无法创建物品: " + shorthand);
    }
    return item;
}
```
