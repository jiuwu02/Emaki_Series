# Emaki CoreLib - 项目概述

## 插件简介

**Emaki CoreLib** 是专为 Emaki 系列插件设计的核心工具库，为 Minecraft Paper 服务器提供一套统一、高效的基础功能模块。作为 Emaki 生态系统的基石，CoreLib 封装了 GUI 系统、物品源管理、表达式引擎、条件评估器、操作系统、数学工具和文本处理等核心功能，让上层插件能够专注于业务逻辑实现，而无需重复造轮子。

### 核心价值

- **统一抽象层**：为 Emaki 系列插件提供一致的 API 接口和数据结构
- **模块化设计**：各功能模块独立运作，可按需使用
- **高性能实现**：采用缓存、懒加载等优化策略，确保服务器性能
- **扩展友好**：支持自定义解析器、渲染器和处理器，满足定制化需求
- **声明式配置**：通过 YAML 配置实现复杂业务逻辑，无需编写代码

## 模块架构

```
Emaki CoreLib
├── gui/                    # GUI 系统
│   ├── GuiService          # GUI 服务管理器
│   ├── GuiTemplate         # GUI 模板定义
│   ├── GuiSession          # GUI 会话管理
│   └── GuiTemplateParser   # 模板解析器
│
├── item/                   # 物品源系统
│   ├── ItemSource          # 物品源抽象
│   ├── ItemSourceRegistry  # 物品源注册表
│   └── ItemSourceParser    # 物品源解析器
│
├── expression/             # 表达式引擎
│   └── ExpressionEngine    # 数学表达式计算
│
├── condition/              # 条件评估器
│   └── ConditionEvaluator  # 条件表达式评估
│
├── operation/              # 操作系统
│   ├── OperationRegistry   # 操作注册表
│   ├── OperationExecutor   # 操作执行器
│   ├── OperationContext    # 执行上下文
│   └── builtin/            # 内置操作
│
├── economy/                # 经济系统
│   ├── EconomyManager      # 经济管理器
│   ├── EconomyProvider     # 经济提供者接口
│   └── CoinsEngine/Vault   # 经济桥接
│
├── placeholder/            # 占位符系统
│   ├── PlaceholderRegistry # 占位符注册表
│   └── PlaceholderResolver # 占位符解析器
│
├── math/                   # 数学工具
│   ├── Numbers             # 数值处理工具
│   └── Randoms             # 随机数生成器
│
└── text/                   # 文本处理
    ├── Texts               # 文本工具类
    └── MiniMessages        # MiniMessage 解析器
```

## 快速上手

### 安装步骤

1. **下载插件**：将 `Emaki_CoreLib-x.x.x.jar` 放入服务器的 `plugins` 目录
2. **启动服务器**：首次启动会自动生成默认配置文件
3. **验证安装**：控制台显示 `Emaki_CoreLib enabled.` 即表示安装成功

### 配置文件位置

```
plugins/
└── Emaki_CoreLib/
    ├── config.yml           # 主配置文件
    └── lang/
        └── zh_CN.yml        # 中文语言文件
```

## 开发者须知

### 作为依赖使用

在 `pom.xml` 中添加依赖：

```xml
<dependency>
    <groupId>emaki.jiuwu.craft</groupId>
    <artifactId>emaki-corelib</artifactId>
    <version>1.0.0</version>
    <scope>provided</scope>
</dependency>
```

在 `plugin.yml` 中声明依赖：

```yaml
depend:
  - Emaki_CoreLib
```

### 获取插件实例

```java
EmakiCoreLibPlugin plugin = EmakiCoreLibPlugin.getInstance();
```
